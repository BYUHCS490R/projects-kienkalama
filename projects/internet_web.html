ASCII and Unicode

Unicode was made to be an agreement between all languages on what code represented what information. It is also used by all languages, not just English. It can represent more characters than ASCII because it uses more bits.

What is UTF-8?

UTF-8 stands for Unicode Transformation Format- 8 bit. It is kind of like a translator that is able to speak or represent both Unicode and ASCII. It also made it so that ASCII could use more characters and symbols like emojis.

What is does URl mean?

URL stands for Uniform Resource Locator. The URL is made up of 6 main parts: the Protocol, Domain, Port, Path, Query and Fragment. The computer reads these parts and it instructs it on how to get there.

HTTP

HTTP stands for Hyper text transfer protocol.

HTTPS

HTTPS is the same as HTTP but it is secured using a digital certificate (SSL/TLS)

IP Address

DNS

W3C recomendations

W3C means World Wide Web Consortium. W3C recommendations are the rules and regulations that the World Wide Web Consortium agrees on.
